## 一、必问核心问题（22条/26条/46条版）

### 1. 基本情况（必问！）
- [ ] 做月嫂几年了？
- [ ] 带过多少宝宝？
- [ ] 带过最小的宝宝多大？最大的呢？
- [ ] 带过早产儿吗？双胞胎呢？
- [ ] 最长一单做了多久？最短呢？
- [ ] 上一单为什么下户？

### 2. 专业能力（宝宝护理）
- [ ] **宝宝黄疸怎么观察？** 生理性和病理性怎么区分？
- [ ] **宝宝肠绞痛怎么处理？**
- [ ] **宝宝发烧怎么办？**
- [ ] **宝宝呛奶如何急救？**（海姆立克法）
- [ ] 脐带护理用什么？护理步骤？
- [ ] 宝宝洗澡水温多少度？
- [ ] 宝宝大便什么颜色是正常的？
- [ ] 宝宝湿疹怎么处理？
- [ ] 宝宝红屁股怎么处理？

### 3. 喂养问题（重点！）
- [ ] **母乳喂养还是奶粉喂养？**
- [ ] **你会通乳吗？** ← 热门问题
- [ ] 妈妈没奶怎么办？
- [ ] 宝宝夜奶怎么安排？
- [ ] 混合喂养怎么操作？
- [ ] 乳头混淆怎么办？

### 4. 妈妈护理
- [ ] 产后妈妈伤口怎么护理？（顺产/剖腹产）
- [ ] **堵奶/乳腺炎怎么办？**
- [ ] 月子餐你会做吗？什么菜系？
- [ ] 一天工作流程是什么样的？
- [ ] 产后抑郁你怎么看？
- [ ] 妈妈没食欲怎么办？

### 5. 行为考察（最重要！筛选老油条）
- [ ] **能否接受带娃时不看手机？** ← 热门问题！
- [ ] 晚上谁带宝宝？你睡得沉吗？打呼噜吗？
- [ ] 遇到育儿分歧听谁的？
- [ ] 能和家里老人相处好吗？
- [ ] 空闲时间你做什么？
- [ ] 做饭时能兼顾家里其他人吗？

## 六、必问的考察细节

### 看证书（原件！）
- [ ] 月嫂证、育婴师证
- [ ] 健康证（有效期？是否有传染病）
- [ ] 身份证原件
- [ ] 资质证书原件

### 看经验（追问细节！）
- [ ] 带过最棘手的案例是什么？怎么解决的？
- [ ] 遇到医疗问题怎么处理？（看是否盲目自信）
- [ ] 宝宝生病送不送医院？谁决定？

### 看性格（观察细节！）
- [ ] 沟通风格：强势还是温和？
- [ ] 回答问题是否敷衍？（背答案 vs 真实经验）
- [ ] 眼神是否真诚
- [ ] 说话是否靠谱、严谨

### 看生活习惯（很重要！）
- [ ] 睡眠质量如何？打呼噜吗？← 影响休息！
- [ ] 卫生习惯：洗手频率、指甲长度
- [ ] 饮食偏好：有忌口吗？
- [ ] 做饭口味：清淡还是重油盐？

## 七、避坑指南（来自评论区）

### ❌ 老油条特征（遇到快跑！）
| 特征 | 案例 | 来源 |
|------|------|------|
| 问"能否接受带娃时不看手机" | 回答"有空看见了就回" ← 敷衍！ | 小红书评论区 |
| 到饭点问雇主 | "点外卖还是自己做" | 小红书评论区 |
| 说错小时被纠正 | 反说雇主"胡扯" | 小红书评论区 |
| 不主动 | 什么都不做等指令 | 抖音 |
| 过于自信 | 什么问题都说"没问题" | 通用 |

### ❌ 面试不问=踩坑
1. **不问健康状况** → 必须看健康证！
2. **不问睡眠习惯** → 有的月嫂打呼噜影响休息
3. **不问做饭** → 有的只做宝妈一个人的饭
4. **不问节假日** → 春节、国庆双倍工资？
5. **不问退订政策** → 万一不满意怎么退？
6. **不问上一单下户原因** → 可能有隐情！

### ✅ 神仙月嫂特征
- 回答问题具体、有案例
- 主动问妈妈想吃什么
- 不玩手机、专注宝宝
- 有边界感、不多话
- 眼里有活、主动做事
- 说话严谨、不夸大
